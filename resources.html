前言

在学习渗透测试的过程中，我整理了一些优质的学习资源，分享给大家。

🎯 靶场练习

DVWA 入门

Damn Vulnerable Web Application，最经典的 Web 漏洞练习靶场。包含 SQL注入、XSS、CSRF、文件上传、命令执行等常见漏洞。

访问 DVWA

Pikachu 入门

中文 Web 漏洞靶场，对新手非常友好配有详细的中文说明。

访问 Pikachu

HackTheBox 进阶

国外知名的渗透测试练习平台，有大量真实环境的渗透挑战。

访问 HackTheBox

VulnHub 进阶

提供大量虚拟机镜像，可以本地搭建进行渗透练习。

访问 VulnHub

PortSwigger Web Security Academy 入门

Burp Suite 官方提供的 Web 安全学习平台，免费且质量极高。

访问 Academy

💡 学习路径建议

  • 第一阶段: DVWA + Pikachu 入门，理解常见漏洞原理
  • 第二阶段: 学习工具使用 (Nmap, SQLMap, Burp Suite)
  • 第三阶段: HackTheBox/VulnHub 进阶练习
  • 第四阶段: 参与真实项目，积累实战经验

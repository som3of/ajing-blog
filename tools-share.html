常用渗透测试工具分享

前言

在渗透测试学习过程中，我接触并使用了很多优秀的工具。这里分享一些最常用、也最值得投入时间掌握的工具。

信息收集工具

Nmap

最经典的端口扫描工具，适合做端口、服务版本、NSE 脚本检测。

nmap -sV -p- -T4 target.com

Subfinder

快速、简洁的子域名发现工具。

subfinder -d target.com

httpx

快速识别 Web 服务，支持批量处理。

cat targets.txt | httpx

漏洞扫描工具

Nuclei

基于模板的漏洞扫描工具，适合 CVE、指纹和配置问题检测。

nuclei -u https://target.com -t cves/

SQLMap

自动化 SQL 注入检测和利用工具。

sqlmap -u "http://target.com?id=1"

目录与爆破工具

Dirsearch

Python 编写的目录扫描工具，简单易用。

FFUF

快速、灵活的 Web 模糊测试工具。

Hydra / John

分别适合在线密码测试和离线口令破解。

工具使用心得

  • 组合使用：信息收集 → 漏洞扫描 → 漏洞利用。
  • 自动化：用脚本串联工具，提高效率。
  • 字典很重要：好的字典能让工作事半功倍。
  • 不要过度依赖工具：工具是辅助，理解原理更重要。

🛠 常用渗透测试工具分享

前言

在渗透测试学习过程中，我接触并使用了很多优秀的工具。本文分享一些最常用、感觉最好用的工具。

🔍 信息收集工具

Nmap - 网络扫描神器

最经典的端口扫描工具，必备技能。

  • 端口扫描
  • 服务版本识别
  • 操作系统检测
  • 脚本扫描 (NSE)

nmap -sV -p- -T4 target.com

Subfinder - 子域名枚举

快速、简洁的子域名发现工具。

subfinder -d target.com

httpx - 快速存活检测

快速识别 Web 服务，支持批量处理。

cat targets.txt | httpx

🎯 漏洞扫描工具

Nuclei - 高效 POC 扫描

基于模板的漏洞扫描工具，更新频繁，插件丰富。

  • CVE 漏洞检测
  • 指纹识别
  • 自定义 POC

nuclei -u https://target.com -t cves/

SQLMap - SQL 注入神器

自动化 SQL 注入检测和利用工具。

sqlmap -u "http://target.com?id=1"

📁 目录扫描工具

Dirsearch

Python 编写的目录扫描工具，简单易用。

python -m dirsearch -u https://target.com

FFUF

快速、灵活的 Web 模糊测试工具。

ffuf -u https://target.com/FUZZ -w wordlist.txt

🔓 密码爆破工具

Hydra

支持多种协议的暴力破解工具。

hydra -L users.txt -P passwords.txt target.com ssh

John the Ripper

密码哈希破解工具，支持多种格式。

john hash.txt --wordlist=passwords.txt

🛡️ 综合渗透框架

Metasploit (MSF)

最强大的渗透测试框架，支持漏洞利用、payload 生成等。

  • 漏洞利用
  • 后渗透模块
  • Payload 生成
  • 辅助模块

💡 使用心得

  • 组合使用: 信息收集 → 漏洞扫描 → 漏洞利用
  • 自动化: 使用脚本串联工具，提高效率
  • 字典很重要: 好的字典能让工作事半功倍
  • 不要过度依赖工具: 工具是辅助，理解原理更重要

🛠 工具清单
